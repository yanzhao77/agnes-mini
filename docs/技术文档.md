# Agnes Mini 技术文档

版本: v1.0.0 | Python: >= 3.10 | 许可证: MIT

---

## 1. 项目概览

Agnes Mini 是一个轻量级 Python SDK, 用于对接 Agnes AI API。
它通过分层架构将模型、Provider、Agent 和 CLI 解耦。
内建 Mock 模式, 无需 API Key 即可离线开发。

### 1.1 关键指标

- 源文件数: 25 (src/ 下)
- 测试用例: 124 个, 全部通过
- 代码覆盖率: 70.5%
- ruff lint: 0 错误

### 1.2 技术栈

- Pydantic v2 / Pydantic Settings - 配置与数据模型
- httpx - 异步 HTTP 客户端
- pytest + pytest-asyncio - 测试
- ruff + mypy - 代码规范与类型检查
- typer - CLI 框架

---

## 2. 架构设计

五层架构自底向上:
- 基础设施层: config / exceptions / logger
- Models 层: 数据模型
- Provider 层: 真实 + Mock API 集成
- Agent 层: 业务逻辑 + 编排
- CLI 层: 用户接口

Mock 模式自动切换: AGNES_API_KEY 为空时使用 MockProvider

---

## 3. 异常体系

25 个异常类, 三层继承, 全部继承自 AgnesMiniError。
包括 ConfigError, ProviderError, AgentError, ResourceError 及其子类。

---

## 4. 数据模型层

6 个公共枚举: Role, FinishReason, ImageSize, AspectRatio, VideoStatus, VideoDuration
聊天模型: ChatMessage, ChatRequest, ChatResponse, ChatStreamChunk, ChatHistory
图像模型: ImageRequest, ImageData, ImageResponse, ImageGenerationResult
视频模型: VideoRequest, VideoTaskResponse, VideoQueryResponse, VideoResult

---

## 5. Provider 层

BaseProvider 提供 HTTP 管理和重试机制 (401/429/5xx/超时)。
ChatProvider: POST /v1/chat/completions (agnes-2.0-flash)
ImageProvider: POST /v1/images/generations (agnes-image-2.1-flash)
VideoProvider: POST /v1/videos + GET /v1/videos/{id} (agnes-video-v2.0)
三个 Mock Provider 模拟完整行为。
create_provider() 工厂自动选择真实或 Mock。

---

## 6. Agent 层

- BaseAgent: 抽象基类
- TextAgent: 聊天 + 对话历史
- ImageAgent: 文生图 + 缓存
- VideoAgent: 视频生成 + 轮询
- OrchestratorAgent: 关键词意图识别 + 路由
