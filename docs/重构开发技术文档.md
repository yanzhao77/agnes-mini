#### 3.3.1 错误处理策略

`
PIPELINE_ERROR_MODE = continue_on_error  # 单镜头失败不影响其余
错误记录到 result.errors[] 列表, 最终输出失败镜头清单
图片生成失败: 跳过该镜头, 继续下一个
视频生成失败: 跳过该镜头 (已消耗配额, 不重试)
剧本分解失败: Pipeline 终止 (无输入无法继续)
`

#### 3.3.2 视频节流设计

`python
class VideoRateLimiter:
    def __init__(self, rpm: int = 1):
        self.interval = 60.0 / rpm  # 免费用户: 60s
        self.last_request_time = 0
    async def acquire(self):
        now = time.time()
        wait = self.interval - (now - self.last_request_time)
        if wait > 0: await asyncio.sleep(wait)
        self.last_request_time = time.time()
`

#### 3.3.3 状态持久化

Pipeline 启动时在 output/ 创建 pipeline_state.json, 逐镜头记录:
`json
{ "pipeline_id": "...", "status": "running",
  "shots": [
    { "shot_number": 1, "image_status": "completed", "image_url": "...",
      "video_status": "queued", "video_task_id": "task_...",
      "video_url": null }
  ],
  "errors": [],
  "progress": { "images_done": 6, "images_total": 9, "videos_done": 0, "videos_total": 9 } }
`

### 4.1 Pipeine 数据流 (含节流)

script/lastpiece.md
    |
    v

    v

    |

## 5. 数据模型变更

### 5.1 Shot 模型 (src/models/script.py)

`python
class Shot(BaseModel):
    shot_number: int            # 镜头编号
    description: str            # 镜头描述
    scene_type: str             # 极远景/特写/中景等
    camera_angle: str | None    # 固定/横移/推进等
    duration: float             # 时长(秒)
    sound: str | None           # 声音设计
    dialog: str | None          # 对白
    image_prompt: str           # [新增] 用于 Agnes AI 生图的 prompt
    video_prompt: str           # [新增] 用于视频生成的 prompt
    reference_image_url: str | None = None  # 生成的参考图 URL
    video_url: str | None = None             # 生成的视频 URL
    status: str = "pending"    # pending/generating/completed/failed

class Script(BaseModel):
    title: str                  # 剧本标题
    style: str | None           # 整体风格描述
    shots: list[Shot]           # 镜头列表
    total_duration: float = 0   # 总时长
`

### 5.2 Breaking Change 声明

| 变更 | 旧值 | 新值 | 影响范围 |
|------|------|------|----------|
| Base URL | api.agnesai.com | apihub.agnes-ai.com | 所有 API 调用 |
| 版本号 | v1.0.0 -> v2.0.0 | 主版本升级 | 所有依赖项目 |
| 新增 verify_ssl | 无 | 默认为 True | 生产环境需确认 |

## 6. 剧本分解流程 (ScriptAgent)

### 6.1 Prompt 模板

使用 agnes-2.0-flash, 请求输出严格 JSON (无 markdown 包裹):

`
你是一个视频剧本分解助手。请将以下剧本分解为镜头列表。

输出 JSON 格式 (不要 markdown 包裹, 不要多余文字):
[{
  "shot_number": 1,
  "description": "镜头描述",
  "scene_type": "极远景/特写/中近景等",
  "camera_angle": "固定/横移/推进/跟随",
  "duration": 5.0,
  "sound": "环境音/对白/静默",
  "image_prompt": "用于生图的详细 prompt",
  "video_prompt": "用于视频生成的详细 prompt"
}]
`

### 6.2 解析失败容错

| 失败场景 | 处理方式 |
|----------|----------|
| LLM 输出非 JSON | 重试 1 次, 失败返回错误 |
| JSON 缺少必填字段 | 使用默认值填充 |
| 镜头数 < 预期 | 以实际为准, 记录警告 |
| 完全解析失败 | Pipeline 终止 |
