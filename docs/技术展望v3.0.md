## 3. 关键新增能力

### 3.1 多 Provider 切换
`python
class ProviderRegistry:
    _registry = {}
    @classmethod
    def register(cls, name, provider_class): cls._registry[name] = provider_class
    @classmethod
    def create(cls, name, settings): return cls._registry[name](settings)
`

注册: ProviderRegistry.register("openai", OpenAIProvider)
使用: provider = ProviderRegistry.create("openai", settings)

## 4. 数据流

剧本 -> LLM 分解 (agn / GPT / Claude)
  -> 角色参考图 (agn-image / DALL-E / SD)
  -> 一致性验证 (cosine similarity score)
  -> 低于阈值则重新生成, 通过则继续
  -> 镜头图片 + 镜头视频
  -> 音频 (TTS + MusicGen + SFX)
  -> 视频合成 (转场 + 字幕 + 混音)
  -> 最终 MP4

| 维度 | v2.0 | v3.0 |
