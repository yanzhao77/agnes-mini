## 9. 风险评估与回退方案
| 风险 | 概率 | 影响 | 缓解措施 | 回退方案 |
|---|---|---|---|---|
| API 限流 1 RPM | 高 | 高 | Pipeline 节流 + 进度提示 | 降低镜头数 |
| Token Plan 配额不足 | 中 | 中 | 启动前预检 + 告警 | 分批生成 |
| ScriptAgent 不稳定 | 中 | 高 | prompt + 重试 + fallback | 手动提供 JSON |
| SSL 配置失败 | 低 | 高 | 默认验证, 保留关闭选项 | verify=False |
| API 变更 | 低 | 中 | 集成测试 + 关注更新 | 适配端点 |
| 工期不足 | 高 | 高 | 按 P0/P1/P2 裁剪 | 推迟 P2 到 v2.1 |
裁剪策略:
- Phase 3 延期: 裁剪状态持久化 + 配额预检
- Phase 4 延期: 裁剪 CLI script 命令 (用 Python API 替代)
- Phase 5 延期: 裁剪 E2E 测试, 推迟文档到 v2.1
