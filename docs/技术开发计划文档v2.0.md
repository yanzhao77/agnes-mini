## 8. 风险评估与回退方案

| 风险 | 概率 | 影响 | 缓解 | 回退 |
| API 限流 1 RPM | 高 | 高 | 节流 + 进度提示 | 降低镜头数 |
| 调用量激增 44 次 | 中 | 中 | 文档标明成本 | 选调用 |
| image-to-image 不准 | 中 | 中 | 多图参考+优选 | text-to-image |
| ScriptAnalysisAgent 不稳 | 中 | 高 | prompt+重试 | 手工 JSON |
| 多图需 2.0-flash | 低 | 低 | 注明版本 | 单图参考 |
| 工期不足 | 中 | 中 | 按 Phase 裁剪 | 推迟 P3/P4 |
